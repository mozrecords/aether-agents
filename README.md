# Aether Video-Agent Starter Kit

Autonomous AI Video Generation Toolkit. Generate viral TikToks, Reels, and Shorts with 1-click.

## Features
- **1-Click Scripting:** AI researches trending topics and writes viral scripts.
- **Auto-Rendering:** Automatic voiceover, subtitles, and B-roll alignment.
- **Next.js Dashboard:** Premium UI to manage your video agents.
- **Gemini & OpenAI Integration:** Powered by the latest LLMs.

## Quick Start
1. Clone this repo.
2. Install dependencies: `npm install` and `pip install -r core/requirements.txt`.
3. Set your API keys in `.env`.
4. Run the dashboard: `npm run dev`.

## Directory Structure
- `/app`: Next.js frontend dashboard.
- `/core`: Python rendering engine and AI logic.
- `/public`: Static assets and generated previews.
